Interpolation

Pour interpoler du JS à l'intérieur du template, grâce aux balises moustaches

  • 1 + 1 = {{ 1 + 1 }}
  • String : {{ 'String' }}
  • Caractères réservés : {{ '<, {{, >, ...' }}
  • Exemple de code : {{ chaine }}
  • Message : {{ message }}
  • Name : {{name}}
